# 北极星程序化交易平台
基于JAVA的程序化交易平台  
本项目基于github上另一个知名的同类项目redtorch演化而来，感谢redtorch项目作者的启发。详细文档请参考wiki

### 一键安装（首次环境初始化）
在linux centOS系统下，在root用户下，输入以下命令（过程中可能会卡住一分钟左右）：  

	curl -L https://gitee.com/KevinHuangwl/northstar/raw/master/autobuild.sh | sh

### 项目更新
执行以下命令：

	cd ~/northstar
	git pull
	bash rebuild.sh
